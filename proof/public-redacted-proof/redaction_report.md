# Redaction Report

This public proof packet was generated from a metadata-preserving private copy of the original benchmark evidence. The original evidence tree was not modified.

## Removed From Public Packet

- Terminal stdout/stderr logs.
- Visual evidence records and screen-capture logs.
- Agent answer text and readiness rationales.
- Prompt text and answer keys.
- Exact repository file lists and inspected artifact names.
- Absolute host, VM, and local filesystem paths.
- Network addresses, user names, and raw environment/setup records.
- Private setup labels and mechanism-bearing condition internals.

## Preserved Publicly

- Official run count and run matrix.
- Per-run numeric operational metrics.
- Per-run success/failure status fields.
- Per-run required-artifact presence booleans.
- Public aggregate metrics and headline deltas.
- SHA-256 digests for the private copied metrics and manifest records backing each public row.

## Chain Of Custody

The public packet was derived from a separate private copy of the evidence tree. That private copy matched the original file count at copy time. Raw evidence remains private for audit.
